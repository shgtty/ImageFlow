# ImageFlow

**ImageFlow** は、指定したローカルフォルダや ZIP ファイル内の画像を自動的に収集し、無限スクロールのデジタルサイネージのように滑らかに表示させることができる軽量な Web アプリケーションです。

## ✨ 主な機能

- **滑らかな無限オートスクロール**: デジタルフォトフレームや店舗サイネージのように、数千枚の画像を自動でスムーズに流し続けます。一番下まで到達するとシームレスにループします。
- **無段階の速度調整**: 再生中いつでもキーボードや画面のボタンでスクロールスピードを上下に変更可能です。
- **動的カラムレイアウト変更**: 画像を並べる「列数」をキーボードで瞬時に 1列～10列 まで自由に増減できます。サーバー通信を行わずDOM再利用で即座に配列が切り替わります。
- **ZIP ファイルの直接読み込み対応**: ディレクトリ内の ZIP 圧縮ファイルを自動検知し、ハードディスクを汚さず「メモリ上でのみオンデマンドに解凍・破棄」して超高速に画像をストリーミング表示します。
- **インテリジェントなフィルター機能**: `include.txt`（ホワイトリスト）と `exclude.txt`（ブラックリスト）を使って、特定の文字が含まれるパスの画像だけを厳格に抽出・除外できます。
- **スマートUIオートハイド**: 操作を行わない状態が3秒間続くと画面上のコントローラーがフェードアウトし、画像だけの没入感のあるUIレス表示に切り替わります。

## 🚀 インストールと起動方法

1. **事前準備**: お使いの PC に [Node.js](https://nodejs.org/) がインストールされている必要があります。
2. 本フォルダ内にある `start_server.bat` をダブルクリックして実行するだけです！
3. 自動で初回セットアップとサーバー起動が行われ、既定のブラウザで画像ギャラリー（`http://localhost:8000`）が開きます。

> **💡 Tips**: デスクトップなどに `start_server.bat` の「ショートカット」を作成しておけば、どこからでもワンクリックで素早くサイネージを起動することができます。

## ⌨️ 操作方法（ショートカットキー）

- `↑` / `↓` キー : オートスクロールの速度を加速 / 減速させます。
- `←` / `→` キー : 画像の「列数（カラム数）」を減らす / 増やす（画像を大きく/小さく表示）。
- `Space` キー : スクロールを現在の速度を保持したまま「一時停止 / 再開」します。
- `F` キー : フルスクリーン表示の切り替え（全画面化）を行います。

## ⚙️ 設定ファイルについて

このアプリは、同じ階層にある **3つの設定ファイル (.txt)** を読み込みます。
それぞれをメモ帳などで開いて編集することで、カスタマイズが可能です。（初回起動時に自動生成されます）
※ 各ファイルの行頭に `#` を付けると、その行はコメントとして無視されます。

### 1. `folders.txt`（検索対象リスト）
読み込みたいフォルダのパスやZIP・画像ファイルのパスを1行ずつフルパスで記述します。サブフォルダも含めて自動検索されます。
* 例：`C:\Users\Public\Pictures`
* 例：`D:\Archives\EventPhotos.zip`

### 2. `include.txt`（ホワイトリスト）
ここに書かれた文字列がファイルパスに「含まれている」画像だけを厳選して抽出します。
* 📝 **AND / OR 検索の切り替え可能**
  デフォルトでは「書かれた単語がすべて含まれる（AND検索）」ですが、ファイル内のどこかに `MODE:OR` と1行だけ記述しておくと、「どれか1つでも含まれる（OR検索）」モードに簡単に切り替わります。

### 3. `exclude.txt`（ブラックリスト）
ここに書かれた文字列がファイルパスに「含まれている」画像を強制的に表示対象から外します。ホワイトリストより優先されます。
* 例：`thumbnail` や `icon` と書いておくと、小さなサムネイル画像などを弾くことができます。

---

> **💻 高度な使い方（コマンドライン引数）**
> CLIやバッチファイルから起動する際、独自のテキストファイルを引数として渡すことで日によって設定を使い分けることが可能です。
> `start_server.bat "my_folders.txt" "my_include.txt" "my_exclude.txt"`
