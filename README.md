# ImageFlow

**ImageFlow** は、指定したローカルフォルダや ZIP ファイル内の画像を自動的に収集し、無限スクロールのデジタルサイネージやスライドショーのように滑らかに表示させることができる軽量な Web アプリケーションです。

## ✨ 主な機能

- **2つの表示モード**:
    - **ギャラリーモード**: 無限スクロールで多くの画像を一覧表示します。デジタルフォトフレームや店舗サイネージに最適です。
    - **デュアルビューモード**: 2枚の画像を画面いっぱいに左右に並べて表示します。自動送り（スライドショー）機能付き。
- **インテリジェントなデュアルビュー**:
    - 画像の縦横比を維持しながら、画面いっぱいに大きく表示します。
    - **横長の画像（風景写真など）を検出すると、自動的に1枚のみを大きく表示**するスマートレイアウト機能を搭載。
    - 切り替えアニメーションを排除し、瞬時の画像表示が可能です。
- **滑らかな無限オートスクロール**: ギャラリーモードでは数千枚の画像をスムーズに流し続け、一番下まで到達するとシームレスにループします。
- **ZIP ファイルの直接読み込み**: ディレクトリ内の ZIP を自動検出し、メモリ上でオンデマンドに展開して画像をストリーミング表示します。
- **柔軟なソート機能**: 「ランダム表示」と「ファイル名昇順」をボタン一つで切り替え可能。各モードごとに設定が保持されます。
- **インテリジェントなフィルター**: `include.txt`（ホワイトリスト）と `exclude.txt`（ブラックリスト）で、表示したい画像のみを厳密に抽出できます。
- **状態の永続化**: スクロール速度、列数、表示モード、スライドショーの間隔、ソート順、表示方向（LTR/RTL）などはブラウザを閉じても次回起動時に自動で復元されます。
- **スマートUIオートハイド**: 操作がないと、コントローラーとマウスカーソルが自動的にフェードアウトし、画像に集中できる没入型UIに切り替わります。

## 🚀 起動方法

1. **事前準備**: お使いの PC に [Node.js](https://nodejs.org/) がインストールされている必要があります。
2. 本フォルダ内にある `start_server.bat` をダブルクリックして実行します。
3. 自動でサーバーが起動し、既定のブラウザで `http://localhost:8000` が開きます。

## 🎮 操作方法

### 共通の操作
- `F` キー / **画面ダブルクリック** / **マウスホイール（中央ボタン）クリック**: フルスクリーン（全画面）表示の切り替え。
- `M` キー: ギャラリーモードとデュアルビューモードの切り替え。
- `R` キー: ソート順（ランダム / ファイル名昇順）の切り替え。
- `Space` キー: 動作（スクロールまたは自動送り）の「一時停止 / 再開」。
- `ESC` キー: フルスクリーン時は通常表示に戻り、通常時はウィンドウ（タブ）を閉じます。
- **マウス操作**: 画面上にマウスを動かすと操作パネルが表示されます。

### ギャラリーモード
- `↑` / `↓` キー: スクロール速度の加速 / 減速。
- `←` / `→` キー: 画像の列数（カラム数）の変更（1～10列）。
- `PageUp` / `PageDown` キー: 前のフォルダ / 次のフォルダへ移動（昇順ソート時のみ）。

### デュアルビューモード
- `←` / `→` キー: 1枚ずつ前の画像 / 次の画像へ移動。
- `↑` / `↓` キー / **左右クリック** / **マウスホイール回転**: 前のページ / 次のページへ移動（表示枚数分スキップ）。
- `Home` / `End` キー: 最初の画像 / 最後の画像へ移動。
- `PageUp` / `PageDown` キー: 前のフォルダ / 次のフォルダへ移動（昇順ソート時のみ）。
- `O` キー / **右下の回転ボタン**: 表示方向（左から右 / 右から左）の切り替え。
- 操作パネルの `+` / `-` ボタン: 自動送りの間隔（秒数）を変更。

## 📝 設定ファイル (.txt)

`config` フォルダ内にある設定ファイルを編集することで、表示内容をカスタマイズできます。
※ 行頭に `#` を付けるとコメントとして無視されます。

### 1. `config/folders.txt`
読み込みたいフォルダのフルパスを1行ずつ記述します。サブフォルダも自動検索されます。
* 例：`C:\Users\User\Pictures`
* 例：`D:\Archives\TravelFiles.zip`

### 2. `config/include.txt`
ファイルパスに含まれるべき文字列を記述します。
* `MODE:AND` (デフォルト): 記述した「すべての単語」を含む画像のみ表示。
* `MODE:OR`: 記述した「いずれかの単語」を含む画像を表示。

### 3. `config/exclude.txt`
除外したいパスに含まれる文字列を記述します。ホワイトリストより優先されます（例: `thumbnail`, `temp`）。

---

> **💻 高度な使い方**
> CLIから起動する際、独自のテキストファイルを引数として渡すことが可能です。
> `node server/server.js "my_folders.txt" "my_include.txt" "my_exclude.txt"`
